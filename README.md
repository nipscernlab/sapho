# SAPHO  
**Scalable Architecture for Hardware Optimization**

**SAPHO** is a scalable hardware architecture and toolchain developed by **NIPSCERN**, designed for processor creation, experimentation, and hardware–software co-design.

This repository distributes the **SAPHO installer**, which bundles the core components of the ecosystem, including:

- **YANC toolchain**, with compilers such as `cmmcomp`, `asmcomp`, and related tools  
- **POLARIS**, a modern graphical IDE built on **Monaco Editor**

Using SAPHO, you can:

- Design custom processors using the **CMM programming language**
- Develop and compile code through the **POLARIS IDE**
- Simulate and analyze signal behavior with the **YAWT Wave Tracer**
- Visualize processor structure and datapaths using the **PRISM RTL Viewer**

SAPHO provides an integrated environment for **processor design, compilation, simulation, and visualization**, enabling rapid experimentation and research in hardware optimization and computer architecture for both Windows and Linux.
